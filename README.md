Lua System SP(3)
================

![Screenshot](screenshot.jpg)

Small old homebrew Lua shell written in Lua for PSP

##What's this ?
A homebrew shell.

##Notes
Compatible with PSP Fat and Slim

##Installation Path
ms0:/PSP/GAME/
